# List of species and breeds for use in petshops

[![npm][npm]][npm-url]
[![size][size]][size-url]

## Install

```
npm i -S @zettabrasil/petshop-especies
```

## Usage

```js

  "Canino": [...],

  "Peixe": [...],
  "Réptil": [...],
  "Roedor": [...]
}

pelagens: [...]

portes: [...]
```
